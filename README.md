DMNN Week 1 Project - Online Retail EDA

Project Information
Language: Python 3
Focus: Data Wrangling and Exploratory Data Analysis

How to Run
Dependencies: Install via pip install pandas matplotlib seaborn.
Dataset: Place online_retail.csv in the data/ folder (local only).
Notebook: Run notebooks/week1_Project.ipynb

Key Summary
Unit of Analysis: Customer Level (aggregated by Customer ID).
Data Cleaning: Removed rows with missing IDs, negative quantities (returns), and zero prices.

Outputs: All charts of the notebook are located in the outputs/ folder.
